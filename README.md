# Akoya (akoya)

Akoya is a tokenized data-access network for US financial institutions. It implements the Financial Data Exchange (FDX) API standard, replacing screen-scraping with token-permissioned API access to consumer accounts.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/akoya/refs/heads/main/apis.yml)

## APIs
- **Akoya FDX API** (`https://api.akoya.com`) - FDX-aligned: Accounts, Account Details, Statements, Transactions, Investments, Tax forms, Payments, Customer

Sandbox: `https://sandbox-products.ddp.akoya.com`

## Notes on OpenAPI
Akoya publishes FDX-aligned reference docs but no canonical public OpenAPI URL. The FDX standard at https://financialdataexchange.org/ is the canonical specification.
